LAB: Running your Task

Run your newly created Task in PE

Overview

Now that you’ve created your first Task and uploaded it to the control repo, we want to run it against our test node in PE.

In this lab you will:

  • Inspect information about your Task via Task metadata
  • Run your Task against the lab node group
  • Review Task run information

Video walkthrough

Labs

1

Inspect Task metadata

  1. In the puppet console under Orchestration, choose Tasks
  2. Click Run a task button
  3. Under Task click the dropdown Enter a task to run and type lab::lin_info or lab::win_info and then click on your Task
  4. Click view task metadata

You can see the description from the Task metadata file created in the previous lab now visualised within the console. The Task parameters we specified in our Task metadata also shows up here.

2

Run your Task

  1. Scroll down the page and click the dropdown Select a target type
  2. Choose Node group and then select or type the Node group name Lab Group then Click Select
  3. Click Run task

3

Review Task run information

Once your Task has successfully completed, at the top of the page you will see some key information.

  • Code environment:This environment in puppet corresponds to branches of the same name containing code specific to that environment (for example “production” or “development”).
  • Task name:Name of Task chosen for this run
  • Start time:What time the Task run was initiated
  • Duration: How long it took for the run across all target nodes to complete
  • User:Name of the user that has run the Task
  • Details:Inventory details or parameters passed to Task

Below this information, you can find output information from your Task on your target node.
